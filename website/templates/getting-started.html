Getting Started


Registration
  • Register for an account and confirm your email.
Import your data
There are two methods to import data to be processed:

Use this option if you are a new user importing data for the first time.

  1. Download and run the Zip-MTGO-Logs script. A .zip file containing your GameLog and DraftLog files will be generated.
  2. Click on Processing -> Import GameLogs.
  3. Nagivate to the .zip file created in Step 1 and upload it.
    • Your browser may ask you to confirm your upload.
  4. Click Import. Your files will be processed in the background and a Load Report will be emailed to you upon completion.

Use this option if you have previously used the MTGO-Tracker desktop application and would like to upload your saved data.

  1. Click on Processing -> Import from MTGO-Tracker
  2. Navigate to your /MTGO-Tracker/save/ and upload it.
    • Your browser may ask you to confirm your upload.
  3. Click Import. Your files will be processed in the background and a Load Report will be emailed to you upon completion.

This will add missing records, or apply any manual revisions from your save data. Note that any records added will not have an associated raw file archived and will not be able to be re-processed.



Complete your dataset
Use the Data Cleaning functions described below to fill in information for attributed that do not import manually. Some column attributes (Deck Archetype, Match Format, Match Type) can not be derived from GameLog files and must be revised manually using Revise Record(s).

The Matches.P1/P2_Subarch columns will be set to 'NA' by default after importing. This function try to guess a deck name based on historical decklists. The accuracy can vary (especially for matches where fewer cards were played). It is recommended that you do this step manually. Limited decks are simply named according to colors played.

  1. Label all matches with a Format. (Matches with Match.Format = NA will be ignored)
  2. Click on Data Cleaning -> Best Guess Deck Names.
  3. Choose which Matches to apply the change to by selecting Constructed, Limited, or All from the dropdown.
  4. Choose whether to overwrite the Match.P1/P2_Subarch for all records, or only those records with Match.P1/P2_Subarch = NA.

NOTE: Overwriting deck names is not reversible. Don't accidentally waste all the time you spent naming decks!

The Games.Game_Winner column will be set to 'NA' if the game's winner could not be determined. Cycle through affected Games and manually select a Game_Winner based on trailing Game Actions.

  1. Click on Data Cleaning -> Get Missing Game Winners.
  2. For each Game shown, choose the player that won (or Skip to leave Game.Game_Winner = NA).

This will cycle through Matches that have been set to Booster Draft or Cube. The cards played will be compared against cards picked in each Draft to find Applicable Draft_IDs. Match results will automatically be applied to the match result columns in the Drafts Table.

  1. Click on Data Cleaning -> Apply Associated Draft_IDs.
  2. Choose from the list of Applicable Draft_IDs to apply the Draft_ID to the Match (or Skip to leave Match.Draft_ID = NA).

Some columns do not have an associated helper function and must be revised manually.

  1. Go to Tables -> Matches and select the Match or Matches you wish to revise.
  2. Click Revise Record(s) and one of two windows will pop-up depending on whether one or multiple records were selected.
    • Use Ctrl-Click or Shift-Click to select multiple records.

NOTE: Overwriting deck names is not reversible. Don't accidentally waste all the time you spent naming decks!

Selected row(s) in the 'Matches' table can be removed from your database. All associated Games and Plays data will also be removed.

  1. Go to Tables -> Matches and select the Match or Matches you wish to revise.
  2. Click Remove Record(s).
  3. Choose whether or not you want to ignore the selected Matches. Ignored Matches will not be included in future imports.
    • A good example of a case when you would remove but not ignore is if you want a GameLog file re-processed.
    • Ignored GameLogs can be unignored from your Profile page.


Explore your dataset
After your data has been processed, imported, and cleaned, you can explore your data and generate insights.

Use the Tables menu to view your data as it is stored in the database. There are two (2) top-level tables that are viewable (Matches, Drafts), assuming those tables contain data.

  1. Click on Tables -> Select a Table.

Double-click on a record to drill-down into the sub-table.

  1. Matches -> Games
  2. Games -> Plays
  3. Drafts -> Draft Picks

There are pre-built dashboards available showing some basic analysis and descriptive statistics.

  1. Click on Dashboards -> Choose a Dashboard.
  2. Apply filters:
    • Change filters using the dropdown menus.
    • Click Generate.

Export your data as a .zip file containing every existing table as individual .csv files. After exporting, use your data to do further data analysis or create custom dashboards and visualization using other tools such as R/Python, Excel, or Tableau.

  1. Click on Processing -> Export to CSV.


Continue importing data as you play more matches on MTGO!
