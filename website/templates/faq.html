Frequently Asked Questions


No! You can import your saved data from MTGO-Tracker.

The preferred method to accomplish this is to upload your files normally (Using Import GameLogs), and then using Import from MTGO-Tracker to apply your manual revisions.

GameLog files do not include any timestamp data in the file contents, so the parsing process relies on file modification date to record the date and time a match was played. This metadata is not preserved when individual files are uploaded, but is preserved when zipped files are uploaded.

The script is scanning your PC for files that match the MTGO Game/DraftLog file naming conventions and filetype. If it finds a Game/DraftLog, it copies it and adds it to a .zip file that can be uploaded to MTGO-DB. The script code can be viewed here.

Deck names and sample decklists are scraped from MTGGoldfish tournament results monthly. The list of cards played during a given match is compared against sample decklists collected over the prior month and the list that best matches is chosen at the 'best guess'.

If MTGGoldfish does not standardize their deck names, it is possible to get similar lists with different names. Also, if very few cards are played during a match, it is very easy to get a mismatch.

This is a difficult function to implement with high accuracy, and is meant to be used as a 'first pass' and should be verified.

MTGO GameLog files do not track cards in-hand. Win rate data by card is available for cards that are cast/played.

Gameplay statistics for MTGO are not offered to this level of granularity (down to the individual play) anywhere else. This software also automates much of the data ingestion process.

Your first import will take the longest, assuming you have a large backlog of data to be processed.

Individual matches only take 1-2 seconds to process, but if you have a backlog of 1000 matches, that could take 20-25 minutes. Once a match has been processed, it will be skipped in future imports and the process will be much faster.

After files are uploaded, they will be scheduled for processing and you can navigate away from the page. A Load Report will be emailed upon completion.

Logs from multiple accounts can be processed normally, but the data will not be combined within the software.

Changing your MTGO Username on your profile page will change the data being shown to you throughout the site.

Currently, only 2-player formats are supported. Commander GameLog files will be skipped by default during processing.
